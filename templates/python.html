Python was invented by Guido van Rossum and released in 1991. Python is an interpreted scripting language with dynamic typing. Developers enjoy using Python due to its readability (though you do need to pay attention to indentation), flexibility, and its comprehensive standard library.

Python is one of the most popular programming languages. Developers can use it for general purpose scripting, web development, scientific computing, artificial intelligence and machine learning, graphical application development (both desktop and mobile), as well as game development. Python is popular in education due to its more beginner-friendly syntax. The Raspberry Pi project adopted Python as their main programming language for its users.
